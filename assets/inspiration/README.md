Drop reference screenshots here when you want the skin UI to borrow a mood, control style, or layout idea.

Recommended workflow:

1. Add 1-5 images to this folder.
2. Give each file a name that hints at the part you like, for example:
   - `retro-floating-widget.png`
   - `glass-operator-panel.png`
   - `compact-icon-buttons.png`
3. If helpful, add one short note beside the image in a matching `.txt` file:
   - `retro-floating-widget.txt`
   - `glass-operator-panel.txt`

Useful note prompts:

- What should we borrow?
- What should we avoid?
- Is this about layout, buttons, typography, color, or motion?

That lets us build a moodboard from your references without copying any one app too literally.
